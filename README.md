# SP500-Data-Filter
This Python project processes historical S&amp;P 500 market data from a CSV file. It parses each row into structured objects, filters entries based on the closing price, and displays the results in a formatted table. The program also supports creating a SQLite database, inserting the original dataset, and generating the filtered table.

**Note:** This project was created for academic and learning purposes. It is not intended for production use.
